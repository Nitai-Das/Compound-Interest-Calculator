Compound Interest Calculator

Overview

This project is simple web-based Compound Interest Calculator. It let users calculate their estimated future value of their investment based on their investment principal amount, annual interest rate and years of keeping the investment

Features
-	User-friendly interface
-	Input validation ensuring all field are filled
-	Compound Interest Calculation
-	Resetting inputs and results

How it Works
1.	A user types in their inputs (principal amount, interest rate, years)
2.	The program would then use the compound interest formula (A = P(1 + r)^t)
3.	Then display the total amount.

Technologies
-	HTML
-	CSS
-	JavaScript

Installation
1.	Clone the repository: https://github.com/Nitai-Das/Compound-Interest-Calculator.git
2.	Open the project folder
3.	Open “index.html” and run the project
